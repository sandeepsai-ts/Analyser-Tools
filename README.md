# Testsigma Analyzer
AI-powered test execution analyzer for Testsigma step results.
## Usage
1. Download testsigma_analyzer.html from this repo
2. Run locally: cd ~/Downloads && python3 -m http.server 8765
3. Open: http://localhost:8765/testsigma_analyzer.html
## Testsigma API Key
Settings > API Keys in Testsigma (Admin/Super Admin role required).
Used as the Bearer token in all API calls.
## Feedback
Use the Feedback tab inside the tool to submit GitHub Issues directly.
